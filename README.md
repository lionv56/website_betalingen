# Website Betalingen (Shopping Cart Demo)

Dit project is een demo webshop frontend met winkelwagenfunctionaliteit.
Gebruikers kunnen producten toevoegen aan hun winkelwagen en een totaalprijs berekenen voordat ze doorgaan naar betaling.

## Features

- Dynamische shopping cart
- Product toevoegen en verwijderen
- Aantal aanpassen per product
- Automatische totaalprijs berekening
- LocalStorage opslag van cart data
- Success en Cancel payment pagina's
- Responsive layout met CSS

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage API

## Project Structure

index.html – webshop pagina  
success.html – betaling succesvol pagina  
cancel.html – betaling mislukt pagina  
style.css – styling  
main.js – shopping cart logica
